L01 BI
Nome Business Intelligence
Noção Processos de organização, coleta, análise, monitoramento e compartilhamento das informações que são a base da gestão de negócios
Classificação Objeto
Impacto(s) O gestor precisa usar o business intelligente para gerir o negocio.
Sinônimo(s) inteligência de negócios
L02 API
Nome Application Programming Interface
Noção Conjunto de rotinas e padrões estabelecidos por um software para a utilização das suas funcionalidades por aplicativos que não pretendem envolver-se em detalhes da implementação do software
Classificação 0bjeto
Impacto(s) O software precisa consumir API's para realizar a BI.
Sinônimo(s) Interface de Programação de Aplicações
L03 CrossData
Nome CrossData
Noção Tipo de dados coletados observando muitos sujeitos
Classificação Objeto
Impacto(s) Os dados coletados das API's serão armazenados dentro do CrossData para possibilitar a fonte de dados das analises.
Sinônimo(s) Dados cruzados, Dados transversais
L04 Metabase
Nome Metabase
Noção Plataforma para compartilhar dados e análises. Um servidor de Business Intelligence de código aberto.
Classificação Objeto
Impacto(s) Os dados passaram pelo metabase para que seja feito as analises dos dados
Sinônimo(s) usuário, usuarios, usuário, usuarios, user, users.
